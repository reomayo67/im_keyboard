# im_keyboard

 33% ergonomic layout Keyboard. Atmega32u4 powered

HAS NOT BEEN ASSEMBLED YET

ALOT of this is heaviliy inspired by ScottoKeebs, I am using his footprint and symbol libraries in kicad, so shoutout to him. He was a huge help in making this possible. [ScottoKeebs](https://github.com/joe-scotto/scottokeebs) Huge inspo from [Plaid](https://github.com/hsgw/plaid)

So I just wanted to make an intro into the hardware engineering space with a simple pcb mkb. This is my first ever officially documented project, so yea. Ive been wanting to make one for a while, and eventually want to evolve into making abuncha wacky ergo keyboards and other stuff like that.

this keyboard uses only through hole components and relies on a devboard. Any type of mx switch can be used, it is not hot swap ready though.
